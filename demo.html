HSP 感知引擎架構

點擊模組，探索引擎如何將文字轉化為多感官體驗

輸入子系統

HSP 引擎感知世界的起點，從多個維度接收資訊：

  • 文字解析：讀懂文本，自適應智慧分段（sentences / paragraphs / adaptive）
  • 語音辨識 + 韻律分析：分析語氣情感，不只聽懂內容
  • 生物特徵感測：穿戴裝置即時讀取心跳等生理訊號
  • 多語言支援：中、英及混合語言文本處理

核心處理：HSP 引擎

系統的心臟與大腦，將資訊轉化為多感官訊號：

  • 動態語音生成：根據劇情起伏調整語調（ElevenLabs TTS + gTTS 備援）
  • 情感回饋迴路：偵測疲憊時自動放慢語速或切換音色
  • 觸覺語法映射：標點符號 → 震動，讓皮膚感覺文章節奏
  • 多模態訊號輸出：根據文本情境觸發氣味、震動等訊號

輸出子系統

將引擎處理後的結果，以最沉浸的方式呈現：

  • 聽覺輸出：充滿情感與溫度的高品質語音播放
  • 觸覺回饋：觸覺背心或手環，讓身體感受故事節奏與情緒
  • 環境模擬：控制氣味產生器，讓書中描寫的味道真實飄散
  • LED 視覺輔助：動態燈光隨情感狀態即時呈現

ModernReader / Semantica 層

統一所有元件通訊的語義框架：

  • Semantica 語言：以語義意圖取代 API 呼叫，零樣板代碼
  • Claw 協議：多 Agent 間的原生通訊，支援量子態傳輸
  • Moltbook 環境：Agent 共享執行空間與持久化狀態管理
  • 量子原語：內建量子模擬，可直接執行 Grover 搜尋等演算法
